🖨️ How to Print / Export PDF

  • Open in Google Chrome (best PDF rendering)
  • Press Ctrl+P / Cmd+P → Destination: Save as PDF
  • Click More settings → set Paper size to Custom: 6.75" × 9.25"
  • Set Margins to None
  • Check Background graphics ✓
  • Scale: 100%
  • Click Save — you'll get 2 pages (Side 1 + Side 2)

⚠ CMYK CONVERSION: This file uses RGB colors. Your print vendor must convert to CMYK before press. Provide this PDF + the color hex values from the guide above.

✓ USPS EDDM compliant: 6.5"×9" trim size (≥ 6.125" long side). Minimum 0.007" thickness required at print.

Blue dashed line = safe zone (keep critical content inside). Red dashed border = bleed edge.
