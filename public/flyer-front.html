🖨️ How to Export Print-Ready PDF

  • Use Google Chrome for best results
  • Ctrl+P / Cmd+P → Destination: Save as PDF
  • Click More settings → Paper size: Custom: 9.25" × 6.75"
  • Margins: None  |  Scale: 100%  |  ✓ Background graphics
  • Save — you'll get a single-page PDF ready for your print vendor

⚠ CMYK: This file uses RGB. Your print vendor must convert to CMYK. Provide hex values: Black #000000, Gold #D4A017, Copper #C0621A

✓ USPS EDDM: 9.0"×6.5" trim (landscape) — longest side 9" ≥ 6.125" minimum ✓

Blue dashes = safe zone. Red dashed border = bleed edge.
